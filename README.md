# 🍕 Pizza Sales Analysis using Python

## 📌 Project Overview

This project is an end-to-end **Pizza Sales Analysis** using Python. The objective is to analyze pizza sales data, uncover business insights, and build meaningful visualizations that help improve business decision-making.

The project demonstrates the complete data analysis workflow, from data cleaning to exploratory data analysis (EDA), business problem solving, and visualization.

---

## 📂 Dataset

The dataset contains detailed pizza sales transactions with the following information:

* Pizza ID
* Order ID
* Pizza Name
* Pizza Category
* Pizza Size
* Quantity
* Unit Price
* Total Price
* Order Date
* Order Time
